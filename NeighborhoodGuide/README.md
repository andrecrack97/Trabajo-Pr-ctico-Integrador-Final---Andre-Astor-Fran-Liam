# Neighborhood Guide (Expo, React Native)

Proyecto base para el TP Final: guía de barrio con video, mapa, ubicación y localization.

## Requisitos

- Node 20.19+ recomendado (Expo SDK 54)
- Expo CLI

## Instalación

```bash
npm install
```

## Ejecución

```bash
npx expo start
```

## Variables de entorno

Crear `.env` (opcional para Google Maps):

```
EXPO_PUBLIC_GOOGLE_MAPS_API_KEY=TU_API_KEY
```

## Funcionalidades

- Tabs: Home, Map, Favorites, Settings
- Stack: PoiDetail
- Video (expo-av), Ubicación (expo-location), Mapas (react-native-maps), Intl (expo-localization)
